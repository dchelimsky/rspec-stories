== DESCRIPTION:

RSpec Stories is a Behaviour Driven Development framework for writing executable code
examples.

== FEATURES:

* Spec::Stories provides a plain text story framework

== INSTALL:

  [sudo] gem install rspec-stories

 or

  git clone git://github.com/dchelimsky/rspec-stories.git
  cd rspec-stories
  rake gem
  rake install_gem
